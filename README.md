## Getting Started

### Installation

1. Clone the repository:

```bash
git clone
cd react-memory-game
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`
